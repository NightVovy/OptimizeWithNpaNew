get_gradient_functions.py: 打印求8偏导结果
=== 目标函数 F 的定义 ===
F = asin((B0*s + E00)/(A0*s + 1)) + asin((B0*t + E10)/(A1*t + 1)) + asin((B1*s + E01)/(A0*s + 1)) - asin((B1*t + E11)/(A1*t + 1)) - pi

=== 8个系数的偏导数公式 (Gradient) ===

--- ∂F / ∂A0 ---
-s*(B1*s + E01)/(sqrt(1 - (B1*s + E01)**2/(A0*s + 1)**2)*(A0*s + 1)**2) - s*(B0*s + E00)/(sqrt(1 - (B0*s + E00)**2/(A0*s + 1)**2)*(A0*s + 1)**2)


--- ∂F / ∂A1 ---
t*(B1*t + E11)/(sqrt(1 - (B1*t + E11)**2/(A1*t + 1)**2)*(A1*t + 1)**2) - t*(B0*t + E10)/(sqrt(1 - (B0*t + E10)**2/(A1*t + 1)**2)*(A1*t + 1)**2)


--- ∂F / ∂B0 ---
s/(sqrt(1 - (B0*s + E00)**2/(A0*s + 1)**2)*(A0*s + 1)) + t/(sqrt(1 - (B0*t + E10)**2/(A1*t + 1)**2)*(A1*t + 1))


--- ∂F / ∂B1 ---
s/(sqrt(1 - (B1*s + E01)**2/(A0*s + 1)**2)*(A0*s + 1)) - t/(sqrt(1 - (B1*t + E11)**2/(A1*t + 1)**2)*(A1*t + 1))


--- ∂F / ∂E00 ---
1/(sqrt(((A0*s + 1)**2 - (B0*s + E00)**2)/(A0*s + 1)**2)*(A0*s + 1))


--- ∂F / ∂E01 ---
1/(sqrt(((A0*s + 1)**2 - (B1*s + E01)**2)/(A0*s + 1)**2)*(A0*s + 1))


--- ∂F / ∂E10 ---
1/(sqrt(((A1*t + 1)**2 - (B0*t + E10)**2)/(A1*t + 1)**2)*(A1*t + 1))


--- ∂F / ∂E11 ---
-1/(sqrt(((A1*t + 1)**2 - (B1*t + E11)**2)/(A1*t + 1)**2)*(A1*t + 1))
