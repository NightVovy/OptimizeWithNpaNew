tilted-chsh的保真度

B_alpha = alpha*<A0> + <A0B0> + <A0B1> + <A1B0> - <A1B1>

maximun quantum violation: sqrt(8+2*alpha^2)
correspond quantum strategy:
reference state: psi_hat = costheta|00> + sintheta|11>
A0_hat = sigma_z
A1 = sigma_x
B0(new) = sigma_z
B1(new) = cos(2mu)sigma_z - sin(2mu)sigma_x
B2(new)

sin2theta = sqrt( (1-alpha^2/4)(1+alpha^2/4) )

alpha = 1 or 0.5 or 0(chsh)


保真度
F = 1/2 * (1 + d_A0B0 + cos(2*theta) * (d_A0 + d_B0))
    + sin(2*theta)/8  * (d_A1B2 - d_A1B0B2B0 - d_A0A1A0B2 + d_A0A1A0B0B2B0)
    - sin(2*theta)/8  * (d_A0A1B0B2 - d_A0A1B2B0 - d_A1A0B0B2 + d_A1A0B2B0)

alpha * d_A0 + d_A0B0 + d_A0B1 + d_A1B0 - d_A1B1 = 【tilted-CHSH】
